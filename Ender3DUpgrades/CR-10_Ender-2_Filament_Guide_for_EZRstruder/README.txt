# Summary

The normal filament guide for the CR-10 and Ender 2 doesn't fit with SeeMeCNC's EZRstruder due to the filament loading path hitting the base of the guide. I've cut some pieces out, so now everything fits nicely. 

Note: having the cables secure underneath is necessary, as the red lever/arm of the EZR would be in the path of the cables if they were secured on top.
