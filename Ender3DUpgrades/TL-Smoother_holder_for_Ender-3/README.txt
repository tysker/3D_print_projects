# Summary

Please use it at your own risk.

UPDATE 12/22/2018
Fixed the problem that type B holder width is 0.1 mm narrow.

UPDATE 10/5/2018
Because there was a problem interfering with parts of the case, we changed the design of Type B.

UPDATE 10/4/2018
Type B was added.
Board size 25.15mm×48.15mm×1.6mm Type A
Board size 28.6mm×49.1mm×1.6mm Type B
I do not have type B board, so I have not confirmed whether it can be used.
Please use the test piece to check if your board can be attached to the holder.
You can save filaments if you can not use them.

V2 UPDATE 9/27/2018
Changed the method of holding the substrate.
